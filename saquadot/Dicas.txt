Para rodar app: 

-> cd saquadot (# No terminal)
-> Criar venv (python -m venv .venv)
-> Ativar Venv (.\.venv\Scripts\activate)
-> Instalar o django (pip install django)
-> Instalar: python -m pip install Pillow (# Para trabalhar com upload e manipulação de imagens)
-> Rodar projeto: python manage.py runserver (# Setifique-se de estar na pasta certa no terminal)
